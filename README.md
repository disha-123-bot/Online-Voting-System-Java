# Online Voting System (Java + MySQL)

## Project Overview
This is a GUI-based Online Voting System developed using Java Swing and MySQL. It allows users to register, login, vote for candidates, and view election results.

## Features
- User Registration & Login
- Secure Voting System
- One User = One Vote
- GUI-based Interface (Java Swing)
- Live Result Display

## Tech Stack
- Java
- Java Swing
- MySQL
- JDBC
- IntelliJ IDEA

## How to Run
1. Clone the repository
   ```bash
   git clone https://github.com/disha-123-bot/Online-Voting-System-Java.git
2. Open in IntelliJ IDEA
3. Configure MySQL database
4. Run `LoginGUI.java`
